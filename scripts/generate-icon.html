Icon Generator

Your app needs an image file, not just code. This tool renders your SVG code into a PNG image.

Instructions:

  1. Click the Download icon.png button below.
  2. Save the file into your project's public folder.
  3. Important: You also need an .ico file for Windows.
  4. Go to ConvertICO.com.
  5. Upload the icon.png you just downloaded.
  6. Download the result as icon.ico and save it to your public folder.
